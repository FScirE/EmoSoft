
To use a Neurosity device:
Create a file in the main folder (located next to this file) called "envNeurosity.env".
This file should contain the following three lines. Keep the quotation marks, and put your Neurosity account & device details inside.
DEVICE_ID="neurosity_device_id_hexadecimal"
EMAIL="neurosity_account@email.com"
PASSWORD="neurosity_password"
